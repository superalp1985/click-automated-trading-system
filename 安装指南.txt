# 交易系统安装指南

## 📋 安装前准备

在使用本安装包之前，您需要先自行安装以下软件：

### 1. MetaTrader 5 (MT5)
- **下载地址**: https://www.metatrader5.com/
- **要求**: 已开通模拟账户或真实账户
- **说明**: 这是交易终端，您的交易策略将通过 MT5 执行

### 2. Ollama (本地大模型运行平台)
- **下载地址**: https://ollama.com/
- **版本要求**: 建议最新版本
- **需要下载的模型**: 根据 config.json 中的配置下载对应模型
  - 常用模型: llama3, mistral, etc.
- **说明**: Ollama 用于本地运行 AI 模型，无需联网即可使用

### 3. Python 环境 (如果尚未安装)
- **下载地址**: https://www.python.org/downloads/
- **版本要求**: Python 3.10 或更高版本
- **必须勾选**: "Add Python to PATH"

### 4. 必需 Python 包
安装以下 Python 包：
```bash
pip install openai metaTrader5 python-dotenv flask
```

## 📦 安装步骤

### 第一步：运行安装程序
1. 双击 `TradingSystem_Setup.exe`
2. 选择安装目录（建议使用默认路径）
3. 点击"安装"等待完成

### 第二步：安装前置软件（如果尚未安装）
按照"安装前准备"章节完成以下软件的安装：
- [ ] MetaTrader 5
- [ ] Ollama + AI模型
- [ ] Python 3.10+
- [ ] Python 依赖包

### 第三步：配置系统
1. 打开安装目录
2. 编辑 `config.json` 文件，配置以下内容：
   ```json
   {
     "mt5_login": "您的MT5账户",
     "mt5_password": "您的MT5密码",
     "mt5_server": "您的经纪商服务器",
     "ollama_url": "http://localhost:11434",
     "model": "您下载的模型名称"
   }
   ```

### 第四步：启动系统
两种启动方式：

**方式一：手动启动（推荐）**
```bash
双击 start_manual.bat
```

**方式二：自动启动**
```bash
双击 start_trading_fixed.bat
```

## ❓ 故障排除

### MT5 连接失败
- 检查 MT5 是否已登录
- 确认 config.json 中的服务器地址正确

### Ollama 连接失败
- 确保 Ollama 正在运行
- 检查 config.json 中的 ollama_url
- 尝试访问 http://localhost:11434

### Python 包缺失
```bash
pip install openai metaTrader5 python-dotenv flask
```

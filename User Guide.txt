## ‚öôÔ∏è Configuration

### Config File Location
- `config.json` in installation directory

### Key Configuration Items

```json
{
  "mt5_login": "123456",        // MT5 account ID
  "mt5_password": "yourpass",   // MT5 password
  "mt5_server": "Broker-Server", // Broker server
  "ollama_url": "http://localhost:11434", // Ollama URL

}
```

### Modifying Configuration
1. Stop the system (close command window)
2. Open `config.json` in Notepad
3. Modify parameters
4. Save file
5. Restart system

## üîç Troubleshooting

### MT5 Connection Failed
**Symptoms**: Web interface shows MT5 not connected

**Solutions**:
1. Check if MT5 is open
2. Check if logged in
3. Verify mt5_server in config.json
4. Restart MT5 and trading system

### Ollama Connection Failed
**Symptoms**: Web interface shows AI not connected

**Solutions**:
1. Ensure Ollama is running (system tray)
2. Check ollama_url in config.json
3. Visit http://localhost:11434 to test
4. Ensure model is downloaded: `ollama list`

### Window Not Found
**Symptoms**: Error "Button position not found"

**Solutions**:
1. Ensure MT5 window title is correct
2. Recalibrate button positions (edit mt5_positions.json)
3. Ensure MT5 window is in foreground

### Slow Response
**Cause**: AI model too large or insufficient PC specs

**Solutions**:
1. Use smaller model (e.g., mistral instead of llama3)
2. Close unnecessary programs
3. Increase computer RAM
